Properties

userId

Introduced in version 1.8

public final String userId

The ID of the user to authenticate in this session

segmentStartDelay

Introduced in version 1.8

public long segmentStartDelay

Milliseconds before the session starts authenticating faces for each pose segment. This gives the user time to assume the requested pose in sessions with liveness detection.

segmentDuration

Introduced in version 1.8

public long segmentDuration

Minimum duration of the pose segment in milliseconds. The authentication can happen very fast. This ensures the process is clear to the user.

isIndefinite

Introduced in version 1.8

public boolean isIndefinite

Set to true for persistent authentication sessions.

requiredNumberOfSegments

Introduced in version 1.8

public int requiredNumberOfSegments

The number of pose segments the user will be required to complete in the session.

showGuide

Introduced in version 1.8

public boolean showGuide

Set to true to show screens that guide the user through authentication before the session starts. Note that only sessions with liveness detection set to REGULAR or STRICT will show the guide.

showResult

Introduced in version 1.8

public boolean showResult

Set to true to show the result at the end of the session.

livenessDetection

Introduced in version 1.8

public VerID.LivenessDetection livenessDetection

The level of liveness detection required in this session.

bearings

Introduced in version 1.8

public EnumSet bearings

Bearings the user may be asked to assume in this session. This is set automatically to EnumSet.of(VerID.Bearing.STRAIGHT, VerID.Bearing.LEFT, VerID.Bearing.UP, VerID.Bearing.RIGHT) for sessions with liveness detection.
